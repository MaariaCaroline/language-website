Apps

Clozemaster an app that allows you to practice vocab in usage.
Learn Korean Korean learning app for K-pop fans. This app provides you with K-pop lyrics translations, K-drama and K-variety shows translations.
Easy Korean News 쉬운 한국어 뉴스 for intermediate learners. This app allows you to read news in Korean and also provides translation for vocab that might be new to you.

Books

1100 Short & Useful Korean Phrases by TTMIK just as the title is saying, this book contains 1100 short & useful Korean phrases with examples.
My Daily Routine in Korean by TTMIK very useful book if you want to learn how to describe every day activities in korean.
Easy Korean Reading for beginners by TTMIK this book contains 30 short stories in Korean. At the end of every story, you can find mini-dictionary and every sentence translated.
Korean Grammar in Use for beginners this book contains all grammar points convered by TOPIK Levels 1 and 2. Also the content is organized by units, which is very helpful.
